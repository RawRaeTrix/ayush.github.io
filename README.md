# My Portfolio

This repository contains the source code for my personal portfolio website, showcasing my skills, projects, and experiences.

## Project Structure

- `index.html`: The main HTML file that serves as the entry point for the portfolio website.
- `css/styles.css`: Contains the styles for the portfolio website, defining the layout, colors, fonts, and other visual aspects.
- `js/main.js`: Includes JavaScript code for any interactive features or functionality on the portfolio website.
- `assets/fonts`: Directory containing font files used in the portfolio website.
- `.github/workflows/deploy.yml`: GitHub Actions workflow configuration that automates the deployment of the portfolio website to GitHub Pages.
- `CNAME`: Specifies a custom domain for the GitHub Pages site, if applicable.

## Setup Instructions

1. Clone the repository to your local machine:
   ```
   git clone https://github.com/yourusername/my-portfolio.git
   ```

2. Navigate to the project directory:
   ```
   cd my-portfolio
   ```

3. Open `index.html` in your web browser to view the portfolio.

## Features

- Responsive design that adapts to different screen sizes.
- Smooth scrolling navigation.
- Interactive elements powered by JavaScript.

## Deployment

This project is set up to be deployed on GitHub Pages. The deployment is automated using GitHub Actions. 

To deploy, simply push your changes to the `main` branch, and the workflow will handle the deployment process.
